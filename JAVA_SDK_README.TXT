THINGWORX JAVA CLIENT SDK README
=================================
The ThingWorx Java Client SDK contains the required Java libraries(JARs) for writing a client in Java that is capable of communicating with the ThingWorx Server.

SAMPLE CODE/DISCLAIMER
======================
NOTE: All sample source code is provided "as-is" and is for reference only.  This may not be used in a production scenario.

In addition to the client SDK libraries, the SDK includes sample Java code based on a test client.  Please use this as a starting point for creating your own client application.
